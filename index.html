💡 Udara di sini dalam kondisi baik. Bantu pertahankan dengan melaporkan pelanggaran merokok!

📡 Bagaimana Cara Kerjanya?

Perangkat ESP32 dipasang di tiang sensor COMFY. Saat pengguna menghubungkan WiFi COMFY-PUBLIC, browser otomatis menampilkan halaman portal ini — tanpa perlu install aplikasi.

🔄 Alur Singkat

1 User konek WiFi COMFY-PUBLIC
2 Browser redirect otomatis ke halaman COMFY
3 Tampil data AQI real-time dari sensor terdekat
4 User bisa lapor pelanggaran KTR dengan mudah
5 Laporan terkirim ke cloud → petugas ditindaklanjuti

⚙️ Komponen Teknis

📡 ESP32 WiFi AP 🌐 DNS Hijacking 🖥️ ESPAsyncWebServer ☁️ HTTPS API 🔐 JWT Auth 📷 Auto Face-Blur
// ESP32 Captive Portal — COMFY System

#include <WiFi.h>
#include <DNSServer.h>
#include <ESPAsyncWebServer.h>

DNSServer dnsServer;
AsyncWebServer server(80);

void setup() {
  // Aktifkan Access Point mode
  WiFi.softAP("COMFY-PUBLIC");

  // Redirect semua DNS ke captive portal
  dnsServer.start(53, "*", WiFi.softAPIP());

  // Serve halaman splash dengan data AQI real-time
  server.on("/", HTTP_GET, [](AsyncWebServerRequest *req) {
    String html = getAQIPage();
    req->send(200, "text/html", html);
  });

  // Handle laporan pelanggaran KTR
  server.on("/submit", HTTP_POST, handleReport);
  server.begin();
}

void loop() {
  dnsServer.processNextRequest();
}
